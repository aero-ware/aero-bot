What can adding AeroBot do for you?

Moderation commands

AeroBot's basic moderation commands are easy to use and reliable. They include ban, kick, mute, warn, and purge. If you seek extremely powerful moderation please use Kali instead.

Reaction Roles

AeroBot has a simple and easy to use system to add reaction roles to ANY message! No need to use a specific message sent by the bot, you can send your own message to use!
