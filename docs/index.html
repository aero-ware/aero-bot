What can adding AeroBot to your server do for you?

Moderation commands

AeroBot's advanced moderation commands allow you to kick, ban (both temporarily and permanently), mute (both temporarily and permanently), warn, and purge messages. More features will be added in the future, and upon request.

Reaction Roles

AeroBot has a simple and easy to use system to add reaction roles to ANY message! No need to use a specific message sent by the bot, you can send your own message to use!
